Summary

This test page demonstrates all nine possible combinations of the image-caption block:

  • Block Position: left, center, right
  • Text Alignment: left, center, right

To use these combinations in your content, format your block declaration as follows:

[image-caption="blockPosition,textAlignment"]

For example: [image-caption="right,left"] creates a right-positioned block with left-aligned text.

This is a test page for the image-caption block. The block supports responsive behavior and will reflow on smaller screens.

To test responsive behavior, resize your browser window or view on a mobile device.
